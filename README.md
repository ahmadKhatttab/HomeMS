Home Energy Management System (HEMS)
This Java-based application is designed to manage and monitor household electricity consumption using an SQLite database.
The system allows users to organize their home into rooms and track various devices, such as ACs, refrigerators,
TVs, and heaters, as well as electric cars.
By applying Object-Oriented Programming (OOP) principles—including inheritance and interfaces—the app calculates
energy usage in kilowatt-hours (kWh) and estimates daily electricity costs based on operating hours.
It provides a practical tool for users to visualize their energy footprint and manage their home devices
efficiently through a simple command-line interface.
